# The Evolution of Web Design: An Interactive Journey

Welcome to a time machine for the World Wide Web. This project is An interactive, scrolling timeline that takes you through the distinct eras of web design, showcasing not just the look, but the *feel* and underlying technology of how the internet has evolved.

## 🕰️ The Eras Explored

*   **1991 - Web 1.0:** The raw, chaotic birthplace. Tables, inline styling, marquees, and "under construction" GIFs.
*   **2004 - The CSS Era:** The structural revolution. `float`, structured divs, standard fonts, and the rise of semantic HTML.
*   **2010 - Web 2.0:** The "App-like" era. Glossy buttons, drop shadows, rounded corners, user-generated content, and the dawn of social networking interfaces.
*   **2016 - Mobile First Era:** Flat design takes over. Responsive layouts, flexbox, grid systems, and utilitarian "SaaS dashboards" designed to work flawlessly on iPhones.
*   **2022 - Modern/Bento Era:** Design elevated. Dark mode by default, glassmorphism, subtle borders, bento grids, and huge typography.
*   **2025 - Intelligent Interfaces (AI Era):** UI becomes ephemeral. Interfaces generated on-the-fly, sentient components, and conversational navigation.
*   **2030+ - The Spatial Web:** The DOM dissolves. Data becomes the environment through 3D canvases, WebGL, and interfaces that respond to physics.

## ✨ Features

*   **Immersive UIs:** Each section doesn't just describe the era; it recreates an interactive "mini-browser" or application true to that specific time period's design language.
*   **Tech Inspectors:** Clickable insight cards in each era break down the dominant Layout techniques, Styling trends (CSS/HTML), and the overall "Vibe" of the period.
*   **Interactive Elements:** Resize the 2016 mobile-first dashboard, drag around the 3D spatial web orbs, or type a message in the 1991 guestbook.
*   **Smooth Scroll Navigation:** Navigate seamlessly through time using the minimalist timeline navigator.

## 🚀 Getting Started

To run this interactive exhibit locally:

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Start the Development Server:**
    ```bash
    npm run dev
    ```

3.  **Explore:** Open your browser to the local URL (typically `http://localhost:3000` or `http://localhost:5173`) and start scrolling!

## 🛠️ Built With

*   **React:** For building complex, stateful UI components representing different eras.
*   **Tailwind CSS:** For rapidly styling the highly diverse requirements of each era (from chaotic Web 1.0 colors to modern glassmorphism).
*   **Motion (Framer Motion):** Essential for the smooth scroll animations, interactive physics (dragging), and dynamic layout changes (like the responsive 2016 view).
*   **Lucide React:** For consistent iconography across the modern eras.
*   **Vite:** For a lightning-fast development experience.
